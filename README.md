## 📌 Project Description
Smart Ticket Deduplicator is an AI-based system that identifies and groups duplicate support tickets using **sentence embeddings** and **similarity matching techniques**.  

It helps in reducing repeated issues, improving ticket resolution speed, and making support systems more efficient.

## 🧠 How It Works
1. User enters support tickets
2. System converts text into embeddings
3. Similarity between tickets is calculated
4. Duplicate tickets are grouped into clusters
5. Output shows grouped ticket categories
